Api Server for Aisland Blockchain

This server can be used to query the transactions by simple https calls.

Query Transactions by Account

You can filter the transaction by account and from date/time to date/time by the following parameters:
  • account - is the account code SS58 format
  • dts - is the date and time of start, in the format YYYY-MM-DD HH:MM:SS.
  • dte - is the date and time of end, in the format YYYY-MM-DD HH:MM:SS.

    • For example:

    https://testnode.aisland.io:9443/transactions?account=5GrwvaEF5zXb26Fz9rcQpDWS57CtERHpNehXCPcNoHGKutQY&dts=2021-08-03+00:00:00&dte=2021-08-20+23:59:59

    Will search the transactions for the account: 5GrwvaEF5zXb26Fz9rcQpDWS57CtERHpNehXCPcNoHGKutQY
    between 03 Agust 2021 time 00:00:00 and 20 August 2021 time 23:59:59.

    "testnode.aisland.io" should be replaced with your domain name.

    Query Single Transaction by Txhash

    You can get the data of a single transaction searching by txhash, for example:

    https://testnode.aisland.io:9443/transaction?txhash=0x91935fe79c429695eb34ca2a191ee2ec7e40b2734691ce7a8ba758c7e1f01e22
